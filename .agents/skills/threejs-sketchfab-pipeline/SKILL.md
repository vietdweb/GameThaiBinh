# Three.js Sketchfab Pipeline

## Description
Sketchfab & GLTF optimization pipeline - asset integration, model traversal, material overriding, PBR fine-tuning, bone mapping, and SkeletonUtils cloning. Use when importing high-quality 3D models from Sketchfab, overriding embedded textures, fixing plastic-looking materials, or cloning animated NPCs without breaking performance.
