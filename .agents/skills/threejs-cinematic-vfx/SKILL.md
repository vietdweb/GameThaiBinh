# Three.js Cinematic VFX

## Description
Cinematic post-processing and visual effects - UnrealBloomPass, Selective Bloom, Chromatic Aberration, Color Grading (LUTs), Motion Blur, and Depth of Field. Use when creating neon glowing items, night-scene atmosphere enhancements, speed-warp blur effects, or making the overall game look like a high-end console title.
